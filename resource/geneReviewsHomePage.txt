inherited conditions in a standardized journal-style format, covering diagnosis,
management, and <a class="def" href="/books/n/gene/glossary/def-item/genetic-counseling/">genetic counseling</a> for patients and their families. Each chapter in
<i>GeneReviews</i> is written by one or more experts on the specific
condition or disease and goes through a rigorous editing and <a href="/books/n/gene/GRrevprocesses/#GRrevprocesses.Peer_Review">peer review</a> process
before being published online.</p><p><i>GeneReviews</i> currently comprises 694 chapters.</p><p>The two general formats for <i>GeneReviews</i> are: chapters focused on a
single <a class="def" href="/books/n/gene/glossary/def-item/gene/">gene</a> or <a class="def" href="/books/n/gene/glossary/def-item/phenotype/">phenotype</a> (~95%) and overviews summarizing genetic causes of common
conditions (e.g., deafness and hearing loss, Alzheimer disease) (~5%).</p><p>To ensure continuing relevant and medically actionable content, each
<i>GeneReviews</i> chapter is <a href="/books/n/gene/GRrevprocesses/#GRrevprocesses.Updates">updated</a> every two to four years (or as
needed) by the author(s) in a formal and comprehensive process curated by the
<i>GeneReviews</i>
<a href="/books/n/gene/GRpersonnel/">editors</a>. Additional <a href="/books/n/gene/GRrevprocesses/#GRrevprocesses.Revisions">revisions</a> may occur
more frequently as needed to reflect significant changes in clinically relevant
information.</p><p>Genetic counseling and testing terms used in
<i>GeneReviews</i> are hyperlinked to definitions in the
<i>GeneReviews</i>
<a href="/books/n/gene/glossary/">Glossary</a>. <a href="/books/n/gene/app5/">Educational Materials</a> provide additional information on key genetics concepts used in <i>GeneReviews</i>.</p><p><i>GeneReviews</i> are indexed in PubMed.</p></div><div><h2>GeneReviews by Title</h2><ul id="toc_tllNBK1116_toc_del1p36" class="simple-list toc"><p><a href="#IX-1">1</a> 揃 <a href="#IX-2">2</a> 揃 <a href="#IX-3">3</a> 揃 <a href="#IX-7">7</a> 揃 <a href="#IX-9">9</a> 揃 <a href="#IX-15">15</a> 揃 <a href="#IX-16">16</a> 揃 <a href="#IX-17">17</a> 揃 <a href="#IX-21">21</a> 揃 <a href="#IX-22">22</a> 揃 <a href="#IX-A">A</a> 揃 <a href="#IX-B">B</a> 揃 <a href="#IX-C">C</a> 揃 <a href="#IX-D">D</a> 揃 <a href="#IX-E">E</a> 揃 <a href="#IX-F">F</a> 揃 <a href="#IX-G">G</a> 揃 <a href="#IX-H">H</a> 揃 <a href="#IX-I">I</a> 揃 <a href="#IX-J">J</a> 揃 <a href="#IX-K">K</a> 揃 <a href="#IX-L">L</a> 揃 <a href="#IX-M">M</a> 揃 <a href="#IX-N">N</a> 揃 <a href="#IX-O">O</a> 揃 <a href="#IX-P">P</a> 揃 <a href="#IX-R">R</a> 揃 <a href="#IX-S">S</a> 揃 <a href="#IX-T">T</a> 揃 <a href="#IX-U">U</a> 揃 <a href="#IX-V">V</a> 揃 <a href="#IX-W">W</a> 揃 <a href="#IX-X">X</a> 揃 <a href="#IX-Y">Y</a> 揃 <a href="#IX-Z">Z</a></p><li id="IX-1"><h3>1</h3></li><li class="half_rhythm" id="toc_itm_NBK1116_toc_del1p36"><a class="toc-item" href="/books/n/gene/del1p36/">1p36 Deletion Syndrome</a><div>Agatino Battaglia.</div><div class="small">Initial Posting: February 1, 2008;
